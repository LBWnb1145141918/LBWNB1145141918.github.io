## ⚠️ 注意事项

### 数据库
- Railway 使用内存数据库（`:memory:`）
- **重启后数据会清空**
- 如需持久化，需配置 Railway 持久化存储

### 环境变量
- `STEAM_API_KEY` 目前是占位符
- 如需启用 Steam 登录，需获取真实的 API Key
- 在 Railway 控制台更新环境变量

### CORS 配置
- 已配置允许 GitHub Pages 域名访问
- 如需添加其他域名，修改 `server.js` 中的 CORS 配置
