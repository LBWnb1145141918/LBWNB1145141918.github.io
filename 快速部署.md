# ⚡ 一键部署 - 3 步更新网站

## 🎯 最简单的部署方法

### Windows 用户

**双击运行** `部署.ps1` 文件

或者在项目中右键 → "使用 PowerShell 运行"

### 或者使用命令行

```bash
# 在项目文件夹中打开 PowerShell，执行：
.\部署.ps1
```

---

## 📋 部署后会发生什么？

1. ✅ 自动检测并添加您的代码更改
2. ✅ 自动提交（带时间戳）
3. ✅ 自动推送到 GitHub
4. ✅ GitHub Pages 自动部署（1-2 分钟）
5. ✅ 全世界都能访问最新网站！
