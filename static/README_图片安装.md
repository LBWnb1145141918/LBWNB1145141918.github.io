# 图片安装说明

## 步骤一：保存图片

将你提供的 "ABYSSAL NETWORKS" 图片保存到以下位置：

```
c:\Users\Administrator\Desktop\LBWnb1145141918.github.io\static\abyssal-logo.png
```

## 步骤二：重命名（如果需要）

- 如果原图片名称不是这个，请重命名为 `abyssal-logo.png`
- 或者保持任意名称，但需要修改 `index.html` 第 323 行的文件名

## 步骤三：刷新网页

保存后刷新网页即可看到效果！

---

## 效果说明

Logo 将会：
- ✅ 显示在终端屏幕正中央
- ✅ 半透明显示（透明度 15%-25%）
- ✅ 带有青绿色发光效果
- ✅ 缓慢呼吸闪烁动画
- ✅ 不会遮挡文字内容
- ✅ 完美融合到终端背景中

## 自定义调整

如果你想调整效果，可以在 `index.html` 第 10-24 行修改：

- `opacity: 0.15` - 调整透明度（0.1-0.5 推荐）
- `width: 60%` - 调整大小（40%-80% 推荐）
- `drop-shadow` 中的颜色 - 调整发光颜色（当前是青色 #00ffcc）
